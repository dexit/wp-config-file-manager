== Description ==

Enable or disable the debug log, update the memory limit in wp-debug.php file.

== Installation ==

1. Install the <code>WP Config File Manager</code> plugin either via the WordPress plugin directory, or by uploading the files to your server at <code>wp-content/plugins</code>.
2. Activate plugin.
3. Open Page/Post and search for "WP Config Manager" block
